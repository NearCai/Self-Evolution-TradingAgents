# Self-Evolution Skill TradingAgents

Agent-Based A-share trading system with Hermes-style self-evolving trading skills.

This repository is a research fork built on the open-source TradingAgents framework. The project adapts the multi-agent trading workflow to China A-shares, reproduces a stable baseline on selected A-share stocks, and adds a skill self-evolution loop inspired by Hermes Agent Self-Evolution.

The retained experiment package focuses on three stocks:

- `600519.SS`: Kweichow Moutai
- `000333.SZ`: Midea Group
- `600036.SS`: China Merchants Bank

The main experiment window is 2026-04-01 to 2026-06-30. The implementation uses long/cash execution, PM-rating decisions, `market,fundamentals` analysts, and DeepSeek flash models for the formal A-share runs.

## Project Layout

| Path | Purpose |
|---|---|
| `tradingagents/` | Core TradingAgents workflow with A-share dataflow, market validation, reporting, and evolution modules. |
| `tradingagents/evolution/` | Trading experience extraction, skill representation, and verifier logic. |
| `scripts/` | Reproduction scripts for baseline backtests, experience building, skill generation, walk-forward validation, weekly online evolution, and plotting. |
| `results/` | Local experiment outputs generated by reproduction commands. This directory is ignored and is not published to GitHub. |
| `reports/` | Local research report source/PDF and report assets. This directory is ignored and is distributed only through the private submission package. |

## Environment

PowerShell setup:

```powershell
conda create -n tradingagents python=3.12
conda activate tradingagents
cd E:\TradingAgents\Self-Evolution-TradingAgents
pip install -e .
$env:PYTHONPATH = (Get-Location).Path
```

Create `.env` from `.env.example` and set the required model or data keys locally. `.env`, `reports/`, and `results/` are intentionally ignored and are not committed to GitHub.

For the A-share experiments, disable noisy US-oriented sources:

```powershell
$env:TRADINGAGENTS_ENABLE_PREDICTION_MARKETS = "false"
$env:TRADINGAGENTS_ENABLE_US_SOCIAL_SOURCES = "false"
```

## Reproduce the Formal Baseline

The canonical baseline command writes local outputs under `results/continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm`. The generated output directory is ignored by Git.

```powershell
python scripts\run_continuous_backtest_ashare.py `
  --tickers 600519.SS,000333.SZ,600036.SS `
  --start-date 2026-04-01 `
  --end-date 2026-06-30 `
  --analysts market,fundamentals `
  --llm-provider deepseek `
  --quick-model deepseek-v4-flash `
  --deep-model deepseek-v4-flash `
  --decision-source pm-rating `
  --memory-mode experiment `
  --memory-holding-days 5 `
  --output-dir results\continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm `
  --disable-prediction-markets `
  --disable-us-social-sources

python scripts\evaluate_continuous_backtest_baselines.py `
  --output-dir results\continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm
```

## Reproduce Self-Evolution Skills

Build April-only experiences and candidate skills:

```powershell
python scripts\build_trading_experiences.py `
  --result-dir results\continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm `
  --start-date 2026-04-01 `
  --end-date 2026-04-30 `
  --output-dir results\walkforward_2026_q2\train_2026_04_experiences

python scripts\generate_trading_skills.py `
  --experience-jsonl results\walkforward_2026_q2\train_2026_04_experiences\trading_experiences.jsonl `
  --output-dir results\walkforward_2026_q2\train_2026_04_skills `
  --missed-upside-return 0.005
```

Run May walk-forward validation with opportunity-focused skills:

```powershell
python scripts\run_continuous_backtest_ashare.py `
  --tickers 600519.SS,000333.SZ,600036.SS `
  --start-date 2026-05-01 `
  --end-date 2026-05-31 `
  --analysts market,fundamentals `
  --llm-provider deepseek `
  --quick-model deepseek-v4-flash `
  --deep-model deepseek-v4-flash `
  --decision-source pm-rating `
  --memory-mode experiment `
  --memory-holding-days 5 `
  --output-dir results\walkforward_2026_q2\val_2026_05_april_opportunity_quality_filtered_skills `
  --evolution-skills-jsonl results\walkforward_2026_q2\train_2026_04_skills\candidate_skills.jsonl `
  --evolution-skill-max-skills 3 `
  --evolution-skill-max-chars 1800 `
  --evolution-skill-types opportunity,promote `
  --disable-prediction-markets `
  --disable-us-social-sources

python scripts\verify_trading_skills.py `
  --gate-preset research `
  --baseline-dir results\continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm `
  --evolved-dir results\walkforward_2026_q2\val_2026_05_april_opportunity_quality_filtered_skills `
  --skills-jsonl results\walkforward_2026_q2\train_2026_04_skills\candidate_skills.jsonl `
  --output-dir results\walkforward_2026_q2\val_2026_05_april_opportunity_quality_filtered_skills\skill_verification_research_gate
```

## Weekly Online Self-Evolution

The supplementary online loop writes local outputs under `results/final_2026_q2_weekly_online_skill_agent`. The retained private artifact covers the completed May weekly windows and is not the main Q2 conclusion; rerunning the command can continue the online loop subject to API availability and cost.

```powershell
python scripts\run_weekly_online_skill_evolution.py `
  --baseline-dir results\continuous_ashare_2026_04_to_06_deepseek_flash_main3_mf_pm `
  --output-dir results\final_2026_q2_weekly_online_skill_agent `
  --initial-skills-jsonl results\walkforward_2026_q2\train_2026_04_skills\candidate_skills.jsonl `
  --tickers 600519.SS,000333.SZ,600036.SS `
  --global-start-date 2026-04-01 `
  --online-start-date 2026-05-01 `
  --end-date 2026-06-30 `
  --analysts market,fundamentals `
  --llm-provider deepseek `
  --quick-model deepseek-v4-flash `
  --deep-model deepseek-v4-flash `
  --evolution-skill-types opportunity,promote `
  --disable-prediction-markets `
  --disable-us-social-sources
```

LLM-driven trading experiments are not fully deterministic. Results can change with model sampling, API behavior, market data availability, and data vendor updates. This repository is for research and course submission only. It is not financial, investment, or trading advice.
